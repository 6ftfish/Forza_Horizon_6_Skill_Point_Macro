# Forza Horizon 6 AFK Skill Point/Money Macro
Hey! I put together this simple AutoHotkey v2 script to fully automate the AFK Super Wheelspin and Skill Point farm in Forza Horizon 6.
This Macro is based on the skill point video by OiAvogadro https://www.youtube.com/watch?v=kI6SW2x1vMg

You earn skill points from compelting the race, you then buy subaru imprezas and use the skillpoints to earn super wheelspins.
Super wheelspins = money

This only works on PC (AFAIK)

How to use the macro:
1. Simply download AutoHotKey v2: https://www.autohotkey.com/ and the script from this repository.
2. Load into the custom game mode, ensuring you know how to use the gamemode, if not. Go watch the youtube video linked above.
3. Right click the script, open with AutoHotkey
4. Ensure you are tabbed into the game, press the 'start' button or the ']' key on your keyboard to start the macro.
4. let it run and carry on with your day job
To stop the macro, press ']' or the stop button.
